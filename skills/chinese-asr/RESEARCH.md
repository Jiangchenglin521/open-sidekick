# 国内语音识别方案调研

## 可选方案

### 1. 百度智能云语音识别
- API: https://ai.baidu.com/tech/speech
- 支持: 普通话、粤语、英语等
- 免费额度: 有

### 2. 阿里云智能语音交互
- API: https://www.aliyun.com/product/nls
- 支持: 实时+离线
- SDK: Python/Java/Go

### 3. 腾讯云语音识别
- API: https://cloud.tencent.com/product/asr
- 支持: 长语音、一句话识别
- 免费额度: 每月5000次

### 4. 科大讯飞 (iFlytek)
- API: https://www.xfyun.cn/services/voicedictation
- 优势: 中文识别率极高
- SDK: 全平台支持

### 5. Moonshot (Kimi) - 音频模型
- API: 已有 kimi 接口，支持音频输入
- 直接对话式识别

### 6. MiniMax
- API: 支持语音对话

### 7. 智谱 AI (GLM)
- API: 部分模型支持语音

## 推荐

最快上手：
1. **腾讯云** - API 简单，免费额度多
2. **百度** - 文档全，SDK 完善
3. **Moonshot** - 大哥已有接入，可能直接可用

## 使用方法

通用流程：
1. 申请 API Key
2. 上传音频文件
3. 获取识别结果
