# 待办识别模式库

## 时间提取模式

### 绝对时间
```
2026-03-21 → 2026-03-21
2026年3月21日 → 2026-03-21
3月21日 → 当年3月21日
```

### 相对时间
```
今天 → 当天
明天 → 次日
后天 → 后天
大后天 → 3天后
```

### 时间段
```
早上8点 → 08:00
上午9点 → 09:00
中午12点 → 12:00
下午3点 → 15:00
晚上8点 → 20:00
```

### 组合示例
```
明天下午3点 → 次日15:00
后天早上9点半 → 后天09:30
本周五下午 → 本周五14:00（默认）
```

## 地点提取模式

```
在xxx → 地点
地点：xxx → 地点
@xxx → 地点
```

## 参与人提取模式

```
跟xxx开会 → xxx
和xxx讨论 → xxx
参与人：xxx, yyy → [xxx, yyy]
```

## 时长提取模式

```
开2小时会 → 120分钟
讨论30分钟 → 30分钟
预计1个半小时 → 90分钟
```

## 完整示例

| 输入 | 解析结果 |
|------|----------|
| 明天下午3点跟王总在会议室A开2小时会 | 时间：次日15:00，时长：120分钟，地点：会议室A，参与人：王总 |
| 下周三早上9点提交报告 | 时间：下周三09:00，标题：提交报告 |
| 今天下午5点前必须给客户发邮件 | 时间：当天17:00，标题：给客户发邮件，优先级：P0（紧急关键词） |
