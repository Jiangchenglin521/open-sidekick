# 优先级规则说明

## 自动优先级判定

系统根据关键词和时间自动计算优先级分数，映射到P0-P3四级。

## 关键词权重表

| 类别 | 关键词 | 权重 |
|------|--------|------|
| **紧急类** | 紧急、马上、立刻、asap、立即、现在、必须今天、急 | +10 |
| **截止类** | deadline、截止、到期、最后期限 | +8 |
| **时间类-近** | 今天、明天、本周 | +5 |
| **时间类-远** | 下周、稍后、有空 | +2 |
| **人物类** | 客户、老板、领导、汇报 | +3 |
| **任务类** | 报告、总结、deadline | +3 |

## 时间紧急度加成

| 截止时间距离 | 加成 |
|--------------|------|
| ≤4小时 | +10 |
| ≤24小时 | +7 |
| ≤72小时 | +5 |
| ≤1周 | +3 |

## 分数映射

| 总分 | 优先级 | 说明 |
|------|--------|------|
| ≥8 | P0-紧急 | 立即处理 |
| 5-7 | P1-高 | 重要且有时间压力 |
| 2-4 | P2-中 | 正常优先级 |
| 0-1 | P3-低 | 可以延后 |

## 示例计算

**输入：** "明天下午跟老板汇报项目进度"

- 明天: +5
- 老板: +4
- 汇报: +3
- 总计: 12 → P0

**输入：** "下周整理会议纪要"

- 下周: +2
- 会议纪要: 0
- 总计: 2 → P2
