# DuerOS EvokeHome

🏠 通过 OpenClaw 控制 DuerOS 智能家居设备

## 简介

DuerOS EvokeHome 是一个 OpenClaw Skill，让你可以通过语音或文字控制小度生态的智能家居设备。

## 功能

- 🔆 灯光控制（开关、亮度调节）
- 📺 电视控制
- 📊 设备状态查询
- ⏰ Token 过期提醒（可选）
- 🔄 自动续期（可选）

## 安装

```bash
openclaw skills install dueros-evokehome
```

## 配置

```bash
evokehome setup
```

## 使用

```
你: 打开台灯
助手: ✅ 已打开台灯

你: 把台灯调到 50%
助手: ✅ 已将台灯亮度调至 50%
```
